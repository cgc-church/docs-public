# Mobile App

The CGC mobile app brings the church to your pocket.

## Download

- **iOS**: [App Store](https://apps.apple.com)
- **Android**: [Google Play](https://play.google.com)

## Features

- Browse and stream sermons
- Offline content downloads
- Push notifications for new content
- Dark mode support
- Bilingual (English & Spanish)
- Secure authentication with Clerk

## Requirements

- **iOS**: 15.0 or later
- **Android**: 10.0 (API 29) or later
- Internet connection for streaming (offline mode available for downloaded content)
