# Sermons

The sermon library is the heart of the CGC platform.

## Browsing Sermons

- **By Preacher**: View all sermons from a specific preacher
- **By Topic**: Filter sermons by category or theme
- **By Date**: Browse chronologically
- **Search**: Full-text search across titles and descriptions

## Streaming & Downloads

- **Stream** audio and video directly in the app or browser
- **Download** for offline access (requires subscription)
- **Playlists**: Create custom playlists of your favorite sermons

## For Administrators

Administrators can manage sermons through the Admin Dashboard:
- Upload audio/video files
- Set titles, descriptions, and metadata
- Schedule publish dates
- Assign preachers and categories
- Manage media files
