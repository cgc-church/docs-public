# Language

## What You Can Do
- Switch the app language between English and Spanish
- Change the language at any time from Settings
- See all menus, labels, and content descriptions in your chosen language

## How It Works

**Scenario: Switching from English to Spanish**
- **Given** I am on the Settings screen
- **When** I tap **Language** and select **Espanol**
- **Then** the entire app interface updates to display in Spanish

**Scenario: Switching from Spanish to English**
- **Given** the app is currently displayed in Spanish
- **When** I go to **Configuracion** (Settings), tap **Idioma** (Language), and select **English**
- **Then** the entire app interface updates to display in English

**Scenario: Viewing content in my chosen language**
- **Given** I have set my language to Spanish
- **When** I browse sermons, menus, and settings
- **Then** all app labels and navigation are shown in Spanish

## What Changes When You Switch Languages

When you change the app language, the following will update:

- All menus and navigation labels
- Button text and form labels
- Settings descriptions
- Notification text
- Error messages and alerts

**Note:** Sermon recordings and some written content may only be available in their original language. The language setting controls the app interface, not the content itself.

### Finding Language Settings After Switching
If you switched to a language you are not familiar with and need to switch back:
1. Tap the **gear icon** to open Settings.
2. Look for the option with a globe or language icon -- this is the Language setting regardless of which language is active.
3. Tap it and select the language you want.

## Common Issues

| Problem | Solution |
|---------|----------|
| The app language did not change | Close the app completely and reopen it. The language change should take effect. |
| Some text is still in the old language | Make sure you are running the latest version of the app. Some older versions may have incomplete translations. |
| I switched to Spanish by accident and cannot find Settings | Tap the gear icon. Then look for **Idioma** (the Spanish word for Language) and select **English**. |
| Sermon content is not in my selected language | The language setting controls the app interface only. Sermon recordings are available in the language they were originally recorded in. |
| The language option is not in Settings | Make sure your app is up to date. The language option is in **Settings > Language** or **Settings > General > Language**. |
