# Admin Dashboard

Welcome to the CGC Admin Dashboard. This section is for church administrators and staff who manage content, users, and settings for the CGC app.

## What You Can Do
- Create, edit, and publish sermons, music, books, videos, and podcasts
- Manage user accounts and assign roles
- Send push notifications to app users
- View and respond to user feedback tickets
- Monitor app activity and content performance

## Who Can Access the Admin Dashboard

The admin dashboard is available to users with one of the following roles:

| Role | Access Level |
|------|-------------|
| **Super Admin** | Full access to all features and settings |
| **Admin** | Manage content, users, and notifications |
| **Editor** | Create and edit content |
| **Media** | Upload and manage media files |

Users with the **Viewer** role do not have access to the admin dashboard.

## How to Access the Admin Dashboard

1. Open a web browser on your computer or tablet.
2. Go to the CGC admin website (your administrator will provide the URL).
3. Log in with your admin credentials (email and password).
4. You will see the admin dashboard with navigation to all available sections.
