# Getting Started

Welcome to Christ Gospel Church's digital platform! Here's how to get started.

## Mobile App

1. **Download** the CGC app from the [App Store](https://apps.apple.com) or [Google Play](https://play.google.com)
2. **Create an account** or sign in with your existing credentials
3. **Browse** sermons, events, and community content
4. **Subscribe** to access premium features

## Web Dashboard (Administrators)

If you're a church administrator:

1. Visit the [Admin Dashboard](https://admin.christgospel.org)
2. Sign in with your administrator credentials
3. If prompted, enter the verification code sent to your device
4. You'll be redirected to the home screen
