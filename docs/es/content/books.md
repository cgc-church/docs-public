# Libros

## Lo Que Puedes Hacer
- Explorar el catálogo completo de libros
- Leer libros directamente en la app
- Resaltar pasajes de texto
- Agregar notas personales a cualquier página
- Marcar páginas con marcadores para regresar después
- Ajustar el tamaño de fuente, estilo de fuente y configuración de pantalla en el lector

## Cómo Funciona

**Escenario: Explorando el catálogo de libros**
- **Dado que** estoy en la pantalla de Libros
- **Cuando** me desplazo por el catálogo
- **Entonces** veo los libros disponibles con su imagen de portada, título y autor

**Escenario: Leyendo un libro**
- **Dado que** estoy viendo la página de detalle de un libro
- **Cuando** toco **Leer**
- **Entonces** el libro se abre en el lector y puedo deslizar o tocar para pasar las páginas

**Escenario: Resaltando texto**
- **Dado que** estoy leyendo un libro en el lector
- **Cuando** mantengo presionada una palabra y arrastro para seleccionar un pasaje, luego toco **Resaltar**
- **Entonces** el texto seleccionado queda resaltado y guardado

**Escenario: Agregando una nota**
- **Dado que** he seleccionado texto en el lector
- **Cuando** toco **Agregar Nota** y escribo mi nota, luego toco **Guardar**
- **Entonces** mi nota se adjunta a ese pasaje y aparece un ícono de nota en el margen

**Escenario: Marcando una página con marcador**
- **Dado que** estoy leyendo un libro
- **Cuando** toco el ícono de **Marcador** en la esquina superior de la página
- **Entonces** la página queda marcada y puedo regresar a ella después desde la lista de marcadores

## Guía Paso a Paso

### Explorar el Catálogo de Libros
1. Abre la app y toca **Libros** desde el menú principal o la pantalla de inicio.
<!-- TODO: Screenshot -->
2. Desplázate por el catálogo para ver los libros disponibles. Cada libro muestra su portada, título y autor.
<!-- TODO: Screenshot -->
3. Usa la barra de búsqueda o los filtros de categoría para encontrar libros específicos.

### Leer un Libro
1. Toca la portada o el título de un libro para abrir su página de detalle.
<!-- TODO: Screenshot -->
2. Toca el botón **Leer** para abrir el libro en el lector.
<!-- TODO: Screenshot -->
3. Desliza a la izquierda o derecha (o toca los bordes de la pantalla) para pasar las páginas.
4. Tu progreso de lectura se guarda automáticamente. Cuando regreses al libro, se abrirá donde te quedaste.

### Resaltar Texto
1. Mientras lees, mantén presionada una palabra para iniciar una selección.
2. Arrastra los controles de selección para cubrir el pasaje que quieres resaltar.
<!-- TODO: Screenshot -->
3. Aparece un menú sobre la selección. Toca **Resaltar**.
4. Elige un color de resaltado si hay opciones disponibles.
<!-- TODO: Screenshot -->
5. Para ver todos tus resaltados, abre el menú del libro y toca **Resaltados**.

### Agregar Notas
1. Selecciona texto en el lector manteniendo presionado y arrastrando.
2. Toca **Agregar Nota** del menú que aparece.
<!-- TODO: Screenshot -->
3. Escribe tu nota en el campo de texto.
4. Toca **Guardar**. Aparece un pequeño ícono de nota en el margen junto al pasaje.
<!-- TODO: Screenshot -->
5. Para ver todas tus notas, abre el menú del libro y toca **Notas**.

### Marcar Páginas con Marcadores
1. Mientras lees, toca el ícono de **Marcador** en la esquina superior de la pantalla.
<!-- TODO: Screenshot -->
2. El ícono se llena para confirmar que la página está marcada.
3. Para ver todos tus marcadores, abre el menú del libro y toca **Marcadores**.
<!-- TODO: Screenshot -->
4. Toca cualquier marcador en la lista para ir directamente a esa página.

### Ajustar la Configuración del Lector
1. Mientras lees, toca el centro de la pantalla para mostrar la barra de herramientas del lector.
2. Toca el ícono de **Configuración** (generalmente un ícono "Aa").
<!-- TODO: Screenshot -->
3. Ajusta las siguientes configuraciones:
   - **Tamaño de fuente** -- Usa el control deslizante o los botones de más/menos para hacer el texto más grande o más pequeño.
   - **Estilo de fuente** -- Elige entre las opciones de fuente disponibles.
   - **Color de fondo** -- Alterna entre los modos de lectura claro, sepia u oscuro.
4. Los cambios se aplican de inmediato. Cierra el panel de configuración para continuar leyendo.

## Problemas Comunes

| Problema | Solución |
|----------|----------|
| El libro no abre | Revisa tu conexión a internet. El libro puede necesitar descargarse antes de poder leerse. |
| Perdí mi progreso de lectura | Asegúrate de tener sesión iniciada en tu cuenta. El progreso de lectura se sincroniza con tu cuenta. |
| Los resaltados o notas desaparecieron | Verifica que hayas iniciado sesión en la misma cuenta. Tus anotaciones se guardan en tu perfil. |
| El texto es demasiado pequeño o grande | Usa la configuración del lector (toca el ícono "Aa") para ajustar el tamaño de fuente. |
| Las páginas son difíciles de leer | Intenta cambiar a un modo de fondo diferente (claro, sepia u oscuro) en la configuración del lector. |
| No puedo seleccionar texto para resaltar | Mantén presionada firmemente una palabra y espera a que aparezcan los controles de selección antes de arrastrar. |
