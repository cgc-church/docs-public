# Notas y Resaltados

## Lo Que Puedes Hacer
- Resaltar texto en libros y transcripciones de sermones con diferentes colores
- Crear notas personales adjuntas a pasajes específicos
- Organizar notas en cuadernos para diferentes temas o estudios
- Agregar etiquetas a las notas para facilitar la búsqueda
- Ver todos tus resaltados y notas en un solo lugar central
- Exportar o revisar tus notas en cualquier momento

## Cómo Funciona

**Escenario: Resaltando texto en un libro**
- **Dado que** estoy leyendo un libro o transcripción de sermón
- **Cuando** mantengo presionada una palabra y arrastro para seleccionar un pasaje
- **Entonces** aparece una barra de herramientas de resaltado y puedo elegir un color de resaltado

**Escenario: Agregando una nota a un resaltado**
- **Dado que** acabo de resaltar un pasaje
- **Cuando** toco "Agregar Nota" en la barra de herramientas de resaltado
- **Entonces** se abre un campo de texto donde puedo escribir mi nota personal y guardarla

**Escenario: Organizando notas en un cuaderno**
- **Dado que** estoy viendo mi pantalla de Notas y Resaltados
- **Cuando** selecciono una nota y toco "Mover a Cuaderno"
- **Entonces** puedo elegir un cuaderno existente o crear uno nuevo para organizar mis notas

**Escenario: Viendo todos los resaltados de un libro**
- **Dado que** estoy leyendo un libro
- **Cuando** toco el ícono de "Notas y Resaltados" en la barra de herramientas de lectura
- **Entonces** veo una lista de todos mis resaltados y notas para ese libro

**Escenario: Buscando notas por etiqueta**
- **Dado que** estoy en la pantalla de Notas y Resaltados en Mi Biblioteca
- **Cuando** toco una etiqueta que creé previamente
- **Entonces** solo se muestran las notas con esa etiqueta

## Guía Paso a Paso

### Resaltar Texto
1. Abre un libro o transcripción de sermón
2. Mantén presionada la primera palabra del pasaje que quieres resaltar
3. Arrastra los controles de selección para cubrir el pasaje completo
4. Aparece una barra de herramientas sobre la selección
5. Toca el **color de resaltado** que prefieras
6. El texto queda resaltado y se guarda automáticamente

### Agregar una Nota
1. Resalta un pasaje como se describió arriba
2. Toca **Agregar Nota** en la barra de herramientas
3. Escribe tu nota en el campo de texto
4. Opcionalmente, agrega una **etiqueta** (como "oración", "estudio" o cualquier palabra que elijas)
5. Toca **Guardar**

### Crear y Usar Cuadernos
1. Ve a **Mi Biblioteca** y toca **Notas y Resaltados**
2. Toca **Cuadernos** en la parte superior
3. Toca **Nuevo Cuaderno** y dale un nombre (por ejemplo, "Sermones del Domingo" o "Estudio Bíblico")
4. Para mover una nota a un cuaderno, selecciona la nota, toca el **menú de tres puntos** y elige **Mover a Cuaderno**
5. Selecciona el cuaderno y toca **Mover**

### Ver Todas Tus Notas y Resaltados
1. Ve a **Mi Biblioteca**
2. Toca **Notas y Resaltados**
3. Explora todas las notas y resaltados de todo tu contenido
4. Usa las opciones de **filtro** para reducir por libro, cuaderno o etiqueta

### Agregar Etiquetas a las Notas
1. Abre una nota existente o crea una nueva
2. Toca el campo de **Etiquetas**
3. Escribe el nombre de una etiqueta y presiona enter
4. Agrega tantas etiquetas como desees
5. Toca **Guardar** para actualizar

## Problemas Comunes

| Problema | Solución |
|----------|----------|
| No puedo seleccionar texto para resaltar | Asegúrate de mantener presionado el tiempo suficiente. Algunos tipos de contenido (como sermones solo de audio) no admiten resaltado. |
| Mis resaltados desaparecieron | Verifica que hayas iniciado sesión en la misma cuenta. Los resaltados se sincronizan con tu perfil. |
| No puedo encontrar una nota que creé | Usa la barra de búsqueda o filtra por etiquetas en la pantalla de Notas y Resaltados. También revisa tus cuadernos. |
| La barra de herramientas de resaltado no aparece | Intenta seleccionar un pasaje más pequeño primero. Si el problema persiste, cierra y vuelve a abrir el contenido. |
| Quiero cambiar el color de un resaltado | Toca el resaltado existente, luego selecciona un nuevo color de la barra de herramientas. |
| Eliminé una nota por accidente | Lamentablemente, las notas eliminadas no se pueden recuperar. Considera crear un respaldo tomando una captura de pantalla de las notas importantes. |
