# Listas de Reproducción

## Lo Que Puedes Hacer
- Crear listas de reproducción personalizadas con tus canciones y sermones favoritos
- Agregar contenido desde cualquier parte de la app a una lista de reproducción
- Reordenar elementos dentro de una lista de reproducción usando arrastrar y soltar
- Renombrar o eliminar listas de reproducción que ya no necesites
- Explorar listas de reproducción destacadas seleccionadas por la iglesia
- Compartir listas de reproducción con amigos y familiares

## Cómo Funciona

**Escenario: Creando una nueva lista de reproducción**
- **Dado que** estoy en la pantalla de Mi Biblioteca
- **Cuando** toco "Nueva Lista de Reproducción" e ingreso un nombre
- **Entonces** se crea una lista de reproducción vacía y está lista para agregar contenido

**Escenario: Agregando una canción a una lista de reproducción**
- **Dado que** estoy escuchando una canción o viendo un sermón
- **Cuando** toco el menú de tres puntos y selecciono "Agregar a Lista de Reproducción"
- **Entonces** puedo elegir una lista de reproducción existente o crear una nueva, y el elemento se agrega

**Escenario: Reordenando elementos en una lista de reproducción**
- **Dado que** estoy viendo una lista de reproducción que creé
- **Cuando** toco "Editar" y arrastro un elemento a una nueva posición
- **Entonces** el orden de la lista de reproducción se actualiza para reflejar mis cambios

**Escenario: Explorando listas de reproducción destacadas**
- **Dado que** estoy en la sección de Listas de Reproducción de Mi Biblioteca
- **Cuando** me desplazo a la sección "Destacadas"
- **Entonces** veo listas de reproducción seleccionadas por la iglesia que puedo explorar y seguir

**Escenario: Eliminando una lista de reproducción**
- **Dado que** estoy viendo una de mis listas de reproducción
- **Cuando** toco el menú de tres puntos y selecciono "Eliminar Lista de Reproducción"
- **Entonces** la lista de reproducción se elimina de mi biblioteca (el contenido original no se elimina)

## Guía Paso a Paso

### Crear una Lista de Reproducción
1. Abre la app y ve a **Mi Biblioteca**
2. Toca la pestaña **Listas de Reproducción**
3. Toca **Nueva Lista de Reproducción**
4. Ingresa un nombre para tu lista de reproducción
5. Toca **Crear**

### Agregar Contenido a una Lista de Reproducción
1. Encuentra una canción, sermón u otro contenido que quieras agregar
2. Toca el **menú de tres puntos** (⋮) junto al elemento
3. Selecciona **Agregar a Lista de Reproducción**
4. Elige una lista de reproducción existente o toca **Nueva Lista de Reproducción** para crear una
5. El elemento se agrega y aparece un mensaje de confirmación
<!-- TODO: Screenshot -->

### Reordenar Elementos de la Lista de Reproducción
1. Abre la lista de reproducción que quieres editar
2. Toca **Editar** en la esquina superior derecha
3. Mantén presionado el control de arrastre junto a cualquier elemento
4. Arrastra el elemento hacia arriba o abajo a su nueva posición
5. Toca **Listo** cuando termines

### Eliminar una Lista de Reproducción
1. Abre la lista de reproducción que quieres eliminar
2. Toca el **menú de tres puntos** (⋮) en la parte superior
3. Selecciona **Eliminar Lista de Reproducción**
4. Confirma tocando **Eliminar**

## Problemas Comunes

| Problema | Solución |
|----------|----------|
| No puedo encontrar mi lista de reproducción | Verifica que hayas iniciado sesión en la cuenta correcta. Las listas de reproducción están vinculadas a tu perfil. |
| No puedo reordenar los elementos | Asegúrate de tocar "Editar" primero. Arrastrar y soltar solo funciona en modo de edición. |
| Una canción que agregué no aparece | El contenido original puede haber sido eliminado de la biblioteca. Intenta buscarlo de nuevo. |
| No veo listas de reproducción destacadas | Las listas de reproducción destacadas requieren conexión a internet. Revisa tu red y desliza hacia abajo para actualizar. |
| Eliminé una lista de reproducción por accidente | Lamentablemente, las listas de reproducción eliminadas no se pueden recuperar. Necesitarás crear una nueva. |
